📖 이 가이드의 학습 방법

  1. 챕터 순서대로 읽기

    챕터 1부터 10까지 순서대로 진행하세요. 뒤쪽 챕터는 앞 챕터의 개념을 전제로 합니다.

  2. 코드 직접 따라 타이핑

    예제 코드를 눈으로만 읽지 말고, 직접 타이핑하면서 외우는 것이 가장 효과적입니다.

  3. 퀴즈로 복습

    각 챕터 끝의 퀴즈를 풀어보세요. 50문제 중 랜덤으로 10문제가 출제됩니다. 점수가 80% 이상 될 때까지 반복하세요!

  4. 실제 프로젝트 코드 분석

    학습 후 직접 프로젝트의 소스 코드를 열어 확인하면 이해도가 크게 높아집니다.
